## 功能特点

- **多网盘支持**: 覆盖市面上所有主流网盘平台
- **操作简便**: 一键粘贴分享链接，快速选择目标网盘
- **实时反馈**: 转存成功或失败有明显的视觉和通知提示
- **时间统计**: 显示转存所需时间，方便了解传输效率
- **历史记录**: 自动保存转存历史，方便查看和管理
- **跨平台**: 同时提供油猴脚本和 Chrome 插件两种安装方式

## 安装方式

### 方式一：油猴脚本（推荐）

1. 首先安装油猴扩展（任选其一）：
   - [Tampermonkey](https://www.tampermonkey.net/) (推荐)
   - [Violentmonkey](https://violentmonkey.github.io/)
   - [Greasemonkey](https://www.greasespot.net/)

2. 安装脚本：
   - 直接打开 `tampermonkey/weiruan-network-disk-transfer.user.js` 文件
   - 或在油猴管理面板中新建脚本，粘贴脚本内容

### 方式二：Chrome 插件

1. 下载本项目的 `chrome-extension` 文件夹

2. 打开 Chrome 浏览器，进入扩展程序页面：
   - 在地址栏输入：`chrome://extensions/`

3. 开启「开发者模式」（右上角开关）

4. 点击「加载已解压的扩展程序」

5. 选择 `chrome-extension` 文件夹

6. 完成安装，可以在工具栏看到插件图标

## 使用说明

### 基本使用流程

1. **复制分享链接**: 从任意网盘复制分享链接

2. **打开转存面板**:
   - 油猴脚本：点击页面右下角的悬浮按钮
   - Chrome插件：点击浏览器工具栏的插件图标

3. **粘贴链接**: 将分享链接粘贴到输入框中

4. **输入提取码**: 如果分享有提取码，请输入（可选）

5. **选择目标网盘**: 点击选择要转存到的目标网盘

6. **设置保存路径**: 输入保存路径（默认为根目录）

7. **开始转存**: 点击「开始转存」按钮

8. **等待完成**: 转存过程中会显示进度和耗时

## 常见问题

### Q: 转存失败怎么办？

A: 请检查以下几点：
1. 确认已登录目标网盘账号
2. 确认分享链接有效且未过期
3. 确认提取码正确（如有）
4. 检查目标网盘空间是否充足

### Q: 为什么转存速度很慢？

A: 转存速度取决于：
1. 网盘服务器响应速度
2. 文件数量和大小
3. 当前网络状况

### Q: 支持批量转存吗？

A: 是的，可以选择多个文件一起转存。

## 注意事项

- 使用前请确保已登录相关网盘账号
- 部分网盘可能有每日转存次数限制
- 大文件转存可能需要较长时间
- 请遵守各网盘的使用条款
